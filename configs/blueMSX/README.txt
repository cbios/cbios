C-BIOS machine configurations for blueMSX
=========================================

Installation instructions:
1. Copy "MSX - C-BIOS" and "MSX2 - C-BIOS" subdirectories to "Machines" in
   the blueMSX installation directory.
2. Copy "cbios_main.rom" and "cbios_sub.rom" into the "MSX - C-BIOS" and
   "MSX2 - C-BIOS" directories (MSX1 doesn't need the sub rom).
