This software is a substitute of BIOS which is needed
when running MSX emulator.
It is targeted for running cartridge image.
Never use without reading LICENSE.


* USAGE

Combine files with running SETB.BAT.
Then you will found CBIOS.ROM is generated.
Specify it as BIOS on emulator or rename to MSX.ROM.

When you boot with pressing SHIFT key, it will go into dump-mode.
